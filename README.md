# Food4Mood

*Tired of thinking of what to eat? Try Food4Mood! Food4Mood provides you with food options suitable for your current mood. No matter what mood you're in, Food4Mood has your back!*


**What does this program do?**
Food4Mood is an emotion recognition web page that captures and analyzes facial expressions through the camera on user's devices. Based on the result, the program recommends various food options to boost the userâ€™s mood. Moods that are detectable in this program include: happiness, sadness, anger, surprise, disgust, and neutral.
